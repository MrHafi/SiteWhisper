# SiteWhisper
A WordPress plugin that instantly turns your site content into an AI-powered chatbot — no manual setup required.

SiteWhisper is a lightweight WordPress plugin that adds an AI chatbot to your site. It automatically reads your site pages and answers visitor questions based on your actual content — no manual training, no complicated setup.
Built with PHP, JavaScript, and Open Router  AI API.

Features:
- Auto-reads your WordPress pages as context
- Floating chat bubble on frontend
- Open Router  AI powered responses
- Admin settings panel
- Lightweight — Limited files architecture 
- No third party dependencies
