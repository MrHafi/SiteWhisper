SiteWhisper

A WordPress plugin that turns your site content into an AI-powered chatbot.

SiteWhisper adds a smart floating chatbot to your WordPress site. It automatically reads your selected pages and answers visitor questions based on your actual content — no manual training required. Built with PHP OOP, vanilla JavaScript, and OpenRouter AI API.

Features:
* Auto-reads your WordPress pages as context
* Floating chat bubble with typing animation
* Session-based chat history
* OpenRouter AI powered responses
* Admin settings panel — API key, chatbot name, welcome message
* Secure AJAX with nonce verification
* Clean uninstall — removes all data on delete
